== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **ADD noopener & noreferrer** in the search box and click on search button.
3. Find ADD noopener & noreferrer plugin.
4. Then click on Install Now after that activate the plugin.
